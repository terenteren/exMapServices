# 프로젝트 소개
네이버, 카카오 지도 API를 활용하여 지도 서비스를 활용한 코드 입니다. <br>
본 예제는 인프런에 코로나맵 제작 강의 예제를 따라 작성 되었습니다.

# 기능 소개
- 지도에 목적지 마커 표시
- 마커 정보 표시
- DB에 목적지 마커 등록
- 마커 클러스터링
- 행정구역(도) 표시


# 사용 언어
MongoDB / Mongoose / Node.js / Express / Naver map API / Kakao map API
